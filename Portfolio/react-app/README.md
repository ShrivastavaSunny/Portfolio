Vite + React + Tailwind starter

Files created:
- package.json
- index.html
- src/main.jsx
- src/App.jsx
- src/components/Portfolio.jsx
- src/index.css
- tailwind.config.cjs
- postcss.config.cjs

Quick start:

```bash
cd portfolio/react-app
npm install
npm run dev
```

Static assets like `resume.pdf` and `profile.svg` are referenced from `/assets` in the parent `portfolio/` folder. When running the Vite dev server, copy those files into `public/` or adjust paths accordingly.
